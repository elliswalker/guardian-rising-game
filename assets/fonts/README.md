# Fonts

Place `PressStart2P.ttf` here.

Download from: https://fonts.google.com/specimen/Press+Start+2P
- Click "Download family"
- Extract and copy `PressStart2P-Regular.ttf` into this folder (no rename needed)

The game uses this font via DesignTokens autoload. Without it, Godot falls back to its default font — everything still works, just not pixel-styled.
