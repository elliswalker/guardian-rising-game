## Player Character — The Speaker

> **Design decision:** The player character is The Speaker. Faceless mask + robes = easy palette swap for customization. No facial features to get wrong. Iconic Destiny lore figure who fits the "commander who doesn't fight directly" role perfectly.

![[Speaker_Infobox.jpg]]
![[Speaker_WithGhost.jpg]]
![[Speaker_D1Triumph.jpg]]
![[Speaker_MaskBroken.jpg]]

**Key visual features:**
- Full-face ornate white/gold mask — no face visible at any point
- Long ceremonial robes — darker at hem, lighter at chest
- Elderly bearing but upright posture
- Often seen with hands clasped or gesturing, never holding a weapon
- The broken mask (SpeakersMaskBroken) is a great reference for the mask shape in isolation

**Pixel art notes:**
- Silhouette is robed figure with distinctive mask profile
- Palette swap potential: swap robe color + mask accent color for unlockable looks
- No need to animate a face — mask stays static

## Solar Guardians

### Cayde-6 — Solar Hunter
*Ace of Spades. Golden Gun. The gambler.*

![[Cayde6_IconicPose.jpg]]
![[Cayde6_FinalShape.jpg]]
![[Cayde6_E32014.jpg]]

## Void Guardians

### Crow (Uldren Sov) — Void Hunter
*Shadowshot. Tether. Reformed prince.*

![[Crow_VanguardInbox.png]]
![[Crow_Enforcer.jpg]]
![[Crow_EarlyGuardian.png]]
![[Crow_Unmasked.jpg]]

- Human/Awoken male — pale skin, violet eyes (Awoken trait)
- Hunter class = hood, cloak, lighter plating
- Early Crow: wore a mask to hide his identity + Spider enforcer gear (dark tones, utilitarian)
- Later Crow: Vanguard Hunter gear, more polished, still hooded
- Slim Hunter build, serious expression when unmasked
- Glint (his Ghost) is distinctive — rose-tinted shell

### Saint-14 — Void Titan
*The Violet King. Perfect Titan. Sentinel Shield.*

![[Saint14_Infobox.jpg]]
![[Saint14_Trials.jpg]]
![[Saint14_Tomb.jpg]]
![[Saint14_Gun.jpeg]]

- Titan class = massive frame, heaviest armor of all Guardians

- Carries Perfect Paradox (shotgun) and an Exotic machine gun
- Most imposing silhouette of any Guardian — built like a tank

![[Ikora_D2.jpg]]
![[Ikora_WithGhost.png]]

## Quick Reference — Silhouettes by Class

| Class | Silhouette traits | Body read at small size |
|-------|------------------|------------------------|
| Hunter | Slim, cloak asymmetry, hood, one side heavier | Lean triangle with one flowing edge |
| Titan | Wide shoulders, squared mass, full plating | Rectangle with shoulder wings |
| Warlock | Robed lower half, structured chest, book hip | Bell shape — wide at hem, narrow at head |
| Speaker | Robed, mask-face, no weapon | Robe silhouette with distinctive mask profile |
